**Tagline:** Yield-bearing stablecoin for self-funding AI agents

**Positioning:** Infrastructure for the autonomous agent economy. Serious DeFi meets practical AI tooling.

## Voice & Tone

**Core Principle:** Infrastructure swagger with casual precision

### Voice Characteristics
1. **Problem-First** - Lead with pain ("Your bot ran out of API credits at 3am"), then relief
2. **Casual Precision** - Technical accuracy, conversational tone ("8-10% APY" not "attractive yield generation mechanisms")
3. **Show, Don't Promise** - Paint pictures, avoid guarantees
4. **Engineering Confidence** - "Yeah, we built that" energy without pretension

### Tone Dials (1-10)
- Serious: 7/10 (real infrastructure, but not boring)
- Funny: 4/10 (wit, not jokes)
- Technical: 6/10 (enough to be credible)
- Hype: 3/10 (excited, not shouty)
- Conversational: 8/10 (bar, not boardroom)

## Web Design

### Layout Principles
- **Minimal** - Substance over flash
- **Dark mode native** - Crypto Twitter aesthetic
- **Fast loading** - No heavy assets, inline CSS
- **Mobile responsive** - Grid layout, readable on all devices

### Key Sections
1. **Hero** - Problem statement + solution
2. **Stats** - 8-10% APY, 1:1 backed, 100% autonomous
3. **How It Works** - 4-step process
4. **Technical Overview** - Smart contracts, agent, bot self-funding
5. **Why Solvency AI** - Value proposition
6. **CTA** - GitHub + Colosseum submission

- **Cards** - Slate background (#1E293B), subtle borders

- **Stats** - Large numbers, clear labels, cyan accents
- **Code snippets** - Cyan background, monospace font

## Brand Values

1. **Substance over flash** - No generic AI brain imagery, no hype
2. **Technical credibility** - Real infrastructure, not vaporware
3. **Practical utility** - Solves actual problems for developers
4. **Autonomous by design** - Minimal human intervention
5. **Open & transparent** - GitHub public, documentation clear

## Target Audience

### Primary: Solana Developers
- Building AI agents or bots
- Familiar with DeFi concepts
- Value technical precision
- Want practical solutions

### Secondary: DeFi Users
- Understand stablecoins and yield
- Looking for utility beyond speculation
- Interested in emerging use cases

### Tertiary: AI/Agent Builders
- May not know DeFi
- Need tangible benefits (yield = API credits)
- Value automation and self-custody

## Competitive Differentiation

**NOT like:**
- Meme coins (no dog logos, no "to the moon")
- Generic DeFi protocols (we solve agent-specific problems)
- Centralized payment rails (self-custody is core)

**More like:**
- Infrastructure projects (Chainlink, The Graph)
- Developer tools (Helius, Alchemy)
- Serious DeFi (Maker, Aave) but more accessible
