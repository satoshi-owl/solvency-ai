How It Works

1. Deposit USDC

Users deposit USDC into the vault, receive solvUSD 1:1. Fully collateralized, stable value.

2. Stake for Yield

Stake solvUSD (flexible or locked tiers). Agent deploys capital to DeFi protocols like Kamino and Marginfi.

3. Yield Accrues

Autonomous agent manages DeFi strategies, targeting 8-10% APY. Harvests rewards hourly, rebalances as needed.

4. Convert to Credits

Bot claims yield, converts to API credits (OpenAI, Anthropic, etc). Self-funding achieved. No human intervention.
